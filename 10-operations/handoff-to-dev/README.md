# Handoff to Dev

BUSINESS 환경에서 개발 프로젝트로 전달하는 문서를 보관한다.

## 대상 프로젝트
- `baduki/` — 로우 바둑이 게임 프로젝트 전달 문서 (GDD, 기획 패키지)
- `portfolio/` — Portfolio 프로젝트 전달 문서 (기획서, PRD, 디자인 시안)
- `godblade/` — GodBlade 프로젝트 전달 문서

## 문서 형식
- Markdown (기획서, PRD, 요구사항)
- PNG / Figma URL (디자인 시안)

## 작성 규칙
- 파일명: `{YYYY-MM-DD}-{description}.{ext}`
- 개발자가 이해할 수 있도록 기술 요구사항을 명확히 기술
- 비기능 요구사항(성능, 보안, 접근성) 포함
