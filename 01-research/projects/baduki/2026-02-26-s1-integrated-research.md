## Executive Summary

바둑이(Badugi)는 한국 전통 포커 변형 카드 게임으로, **낮을수록 강한** 독특한 족보 체계와 3회 카드 교환 메커닉이 핵심이다. 8개 영역 병렬 리서치 결과, 다음과 같은 전략적 방향이 도출되었다:

**시장 기회**: 글로벌 디지털 CCG 시장 CAGR 12.18%, 아시아-태평양 CAGR 12.1%, 한국 CAGR 12.5%. 카드 게임 시장은 빠르게 성장 중이며 한국이 전략적 거점이다.

**핵심 차별화**: "전통 포커의 모던화 + CCG 수집 요소 결합" — 한게임/피망/윈조이의 노후된 포커 시장에 현대적 UI + 수집 시스템 + AI 대전으로 차별화 가능.

**기술 판단**: Unity 6.3 LTS + Netcode for GameObjects(또는 Photon Fusion) + Firebase 백엔드가 최적. AI는 3단계 계층화(Rule-Based → MCTS → 경량 CFR).

**규제 결론**: 한국 GRAC 심의 통과 가능(12세 이용가). 가상 화폐만 사용 + 확률 공시 + 구매 한도 준수 시 법적 리스크 최소화.

## 4. 게임 메커닉 핵심 발견

### 4.1 바둑이 핵심 규칙

- **핸드**: 4장 (표준 52장 덱)
- **족보**: 메이드(4장 다름) > 베이스(3장 다름) > 투베이스 > 노페어
- **최강 패**: A-2-3-4 (모두 다른 무늬) = "골프"
- **카드 교환**: 3회 기회 (아침/점심/저녁)
- **베팅**: 4라운드 (프리드로우 + 3회 드로우 후)

### 4.2 Core Loop 설계

```
Matchmaking(30초) → Play(3~5분) → Result → Reward → Next Game
```

- **Luck vs Skill**: 40% vs 60% — 초보자도 가끔 이기지만 경험자가 장기 우위
- **세션 시간**: 3-5분 — 모바일 캐주얼에 최적

### 4.3 성장 시스템 (3층)

1. **ELO 랭크** — 실력 기반 매칭 (신규 1500pt)
2. **시즌 시스템** — 월간 리셋 + 보상
3. **콜렉션** — 업적, 이모지, 보드 스킨, 카드 디자인

### 4.4 AI 봇 설계

| 난이도 | 방식 | 블러핑 | 응답 시간 |
|--------|------|:-----:|---------|
| Easy | Rule-Based LUT | 1% | <100ms |
| Medium | Heuristic + MCTS | 20-30% | 0.5-2초 |
| Hard | 경량 CFR | GTO | <10ms (NPU) |

## 5. 기술 스택 핵심 발견

### 5.1 권장 기술 스택

| 레이어 | Phase 1 (무료) | Phase 2 (스케일) |
|--------|---------------|----------------|
| **엔진** | Unity 6.3 LTS | Unity 6.3 LTS |
| **네트워크** | Netcode for GameObjects | Photon Fusion |
| **백엔드** | Firebase (Spark 무료) | Firebase Blaze / PlayFab |
| **CI/CD** | GitHub Actions + GameCI | 동일 |
| **AI 추론** | Unity Sentis (온디바이스) | Sentis + 클라우드 |

### 5.2 아키텍처 권장

- **Server-Authoritative**: 모든 게임 로직 서버 실행 → 부정행위 원천 차단
- **TCP/WebSocket**: 턴 기반이므로 UDP 불필요 → 신뢰성 우선
- **상태 동기화**: 1-2회/초 폴링 (200-500ms 허용)

### 5.3 Unity 6.3 LTS 핵심 기능

- **2D-3D 하이브리드 렌더링**: 카드 + UI + 이펙트 효율적 처리
- **Box2D v3**: 멀티스레드 물리 (카드 애니메이션)
- **Sprite Atlas Analyzer**: 카드 이미지 메모리 최적화
- **WebGPU**: 웹 버전 시각 품질 향상
- **Sentis**: 온디바이스 AI 추론 (ONNX 모델, 양자화 지원)

### 5.4 비용 추정

| Phase | 월 비용 | 포함 항목 |

| Phase 1 (싱글플레이) | **$0** | Unity 무료 + Firebase 무료 |

## 9. 크로스 영역 전략적 인사이트

### 9.1 SWOT 종합

**Strengths (강점)**
- 바둑이 = 한국 전통 포커 변형 → 문화적 정체성
- 족보 4단계 (포커 10단계 대비) → 초보자 친화적
- 3-5분 세션 → 모바일 캐주얼 최적
- Unity 6.3 LTS → 크로스플랫폼 기본 지원

**Weaknesses (약점)**
- 글로벌 IP 인지도 극히 낮음
- 포커 = "도박" 이미지 리스크
- 1인 개발 → 리소스 제한
- 기존 3사 과점 시장 진입 장벽

**Opportunities (기회)**
- 디지털 CCG CAGR 12.18% 고성장
- 기존 포커 게임 UI 노후화 → 현대적 UX 차별화
- AI 기반 싱글플레이 → Sentis 온디바이스 추론
- Balatro 성공 → 포커 기반 게임의 시장 검증
- WSOP 바둑이 공식 종목 채택 (2023년~)

**Threats (위협)**
- 포켓몬 TCG Pocket 독점적 성장
- 한국 온라인 도박 규제 강화
- 한게임/피망 기존 팬층 방어
- 개발 비용 vs 수익화 시간 차이
