## 1. 디자인 원칙

### 1.1 핵심 원칙

| 원칙 | 설명 | 적용 예시 |
|------|------|---------|
| **즉각적 피드백 (Instant Feedback)** | 모든 유저 액션에 200ms 이내 시각/촉각 반응 | 카드 탭 → 즉시 위로 올라옴 + 글로우 하이라이트 + 햅틱 |
| **정보 최소화 (Progressive Disclosure)** | 현재 턴에 필요한 정보만 노출, 나머지는 접기/숨김 | 베팅 시 액션 버튼만, 교환 시 카드 선택만 표시 |
| **고급감 (Premium Feel)** | 다크 톤 + 미세 펠트/금속 텍스처 + 부드러운 60fps 애니메이션 | Deep Felt Green 테이블, Gold 악센트, 카드 모션 |
| **원핸드 UX (One-Hand Playable)** | 한 손(엄지) 조작으로 모든 게임 플레이 가능 | 하단 액션 바, 카드 탭 교환, 48dp+ 터치 타겟 |
| **점진적 공개 (Step-by-Step)** | 복잡한 정보를 단계별로 노출 | 튜토리얼 5단계, 족보 가이드 바텀시트 접기/펼치기 |

### 1.2 디자인 키워드

- **Modern Premium** — 10년 노후화된 기존 포커 앱과 완전 차별화
- **Card-Focused** — 카드가 시각적 중심, 나머지 UI는 서포트
- **Fluid** — 모든 상태 전환이 부드러운 애니메이션으로 연결
- **Rewarding** — 보상/성장의 시각적 쾌감 극대화 (파티클, 카운트업)

### 1.3 접근성 목표

- **WCAG AA** 준수
- 최소 터치 타겟: **48x48dp** (모바일 게임 권장 기준)
- 색상 대비 비율: 4.5:1+ (텍스트), 3:1+ (그래픽 요소)
- 카드 무늬 구분: **색상 + 아이콘 형태 이중 인코딩** (색맹 대응)
- 턴 타이머: 시각(원형 게이지 + 색상 변화) + 청각(경고음) + 촉각(햅틱) 3중 피드백

### 3.6 게임 고유 컴포넌트

#### 3.6.1 턴 타이머

```
    ┌─────┐
    │ ◔   │  ← 원형 게이지, 시계 방향 감소
    │ 15  │  ← 중앙: 남은 초 (Inter Bold)
    └─────┘
```

| 속성 | 값 |
|------|:--|
| 크기 | 44x44dp |
| 게이지 색상 | 20~11초: `--color-primary`, 10~6초: `--color-warning`, 5~0초: `--color-error` |
| 경고 애니메이션 | 5초 미만: 펄스(scale 1.0 <-> 1.15, 500ms 반복) + 색상 깜빡 |
| 위치 | 현재 턴 플레이어 프로필 옆 |
| 사운드 연동 | 5초 미만: 틱틱 경고음 재생 |

#### 3.6.2 팟 디스플레이

```
┌─────────────────────┐
│  [코인] POT: 1,200   │  ← 중앙 상단, 칩 아이콘 + Inter SemiBold
│   ○○○                │  ← 칩 시각화 (5개 이하: 개별, 6개+: 스택)
└─────────────────────┘
```

#### 3.6.3 드로우 라운드 인디케이터

```
   [아침]━━━━━━[점심]━━━━━━[저녁]
     ●━━━━━━━━━○━━━━━━━━━○
```

| 속성 | 값 |
|------|:--|
| 위치 | 상단 HUD, 팟 디스플레이 좌측 |
| 현재 라운드 | 채워진 원(`--color-secondary`) + 라벨 Bold |
| 완료 라운드 | 채워진 원(`--color-success`) + 체크마크 |
| 미래 라운드 | 빈 원(`--color-text-secondary`) |

#### 3.6.4 베팅 액션 바

```
┌──────────────────────────────────────────────────┐
│ [폴드]   [체크/콜 200]   [레이즈 ━━━●━━━]  [올인]  │
│  빨강       보조           슬라이더        Primary   │
└──────────────────────────────────────────────────┘
```

| 속성 | 값 |
|------|:--|
| 높이 | 72dp + Safe Area (슬라이더 포함) |
| 배경 | `--color-surface` + 상단 블러(backdrop-filter: blur 8dp) |
| 폴드 | Danger 버튼, 좌측 고정 |
| 체크/콜 | Secondary 버튼, 콜 금액 표시 |
| 레이즈 | 슬라이더(드래그 금액 조절) + 확인 Primary 버튼 |
| 올인 | Primary 버튼 (코인 부족 시 자동 노출) |
| 교환 페이즈 | [스테이(Secondary)] [교환 N장(Primary)] 으로 전환 |

#### 3.6.5 이모지 패널 (바텀시트)

```
┌──────────────────────────────┐
│  이모지 선택                   │
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐ │
```
