## 5. RICE 스코어링 기준

### 5.1 Reach (도달 범위, 1-10)

| 점수 | 기준 |
|:---:|------|
| 10 | 전체 유저 100% 영향 (코어 루프, 인증, 셋업) |
| 9 | 주요 유저 80%+ 영향 (매칭, 시즌) |
| 8 | 활성 유저 60-80% 영향 (상점, 광고, 성장) |
| 6 | 활성 유저 30-60% 영향 (이모지, 토너먼트, 일반 매치) |
| 4 | 관리자/운영팀만 영향 (관리자 도구) |
| 2 | 일부 특수 유저만 영향 |

### 5.2 Impact (영향도, 1-3)

| 점수 | 기준 |
|:---:|------|
| 3 | 핵심 KPI 직접 영향 (리텐션, 수익, 코어 루프) |
| 2 | 보조 KPI 영향 (참여도, 만족도, 사회적 연결) |
| 1 | 간접적 영향 (편의성, 브랜드) |

### 5.3 Confidence (확신도, %)

| 범위 | 기준 |
|:---:|------|
| 90% | 유사 사례 검증됨, 기술적 확실 |
| 80% | 높은 확신, 미미한 불확실성 |
| 60% | 중간 확신, 검증 필요 (신기술, 미검증 시장) |
| 40% | 낮은 확신, 실험 필요 |

### 5.4 Effort (노력, 인주 단위)

인주(person-week) 기준. **1인 개발자 + Unity AI 도구** 생산성 반영.

| 인주 | 기준 |
|:---:|------|
| 1 | 1주 이하 (XS~S 태스크) |
| 1.25 | 약 1.5주 (S~M 태스크, AI 어시스트) |
| 1.5 | 1.5~2주 (M 태스크) |
| 2 | 2주 (M~L 태스크) |
| 3 | 3주 (L 태스크) |
| 4+ | 4주+ (XL 태스크) |

### 5.5 RICE 공식

```
RICE Score = (Reach x Impact x Confidence) / Effort
```
