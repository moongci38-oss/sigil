# 파일명 규칙

## 기본 형식
```
{YYYY-MM-DD}-{description}.{ext}
```

## 예시
```
✅ 2026-02-16-market-analysis-saas.md
✅ 2026-02-16-pricing-model-v2.xlsx
✅ 2026-Q1-marketing-report.md

❌ marketAnalysis.md        (날짜 없음)
❌ 2026_02_16_report.md    (언더스코어 사용)
❌ analysis-02-16.md       (연도 누락)
```
